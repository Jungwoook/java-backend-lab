# java-backend-lab

Java와 Spring 백엔드 학습 내용을 정리하는 스터디 저장소입니다.

문서는 Markdown으로 정리하고, 개념을 코드로 확인해야 할 때는 `examples` 아래에 예제 코드를 추가합니다.

## Study Roadmap

전체 학습 흐름과 문서 작성 방식은 [docs/study-roadmap.md](docs/study-roadmap.md)에서 관리합니다.

1. Java 기초
2. 객체지향 설계
3. DI / IoC
4. Spring Core
5. Spring Boot
6. MVC
7. JPA
